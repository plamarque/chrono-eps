Politique de confidentialité — Chrono EPS

Dernière mise à jour : 21 février 2026

1. Responsable et objectif

Chrono EPS est une application destinée aux enseignants d'EPS pour chronométrer les courses d'une classe et conserver les performances. Cette politique de confidentialité décrit de manière exhaustive comment l'application accède aux données, les recueille, les utilise et les partage.

2. Données traitées

L'application enregistre localement sur votre appareil les données suivantes :

  • Les noms ou identifiants des élèves (données personnelles)
  • Les courses et leurs paramètres (dates, groupes, couleurs)
  • Les passages et temps enregistrés
  • Les performances calculées (temps total, nombre de tours)

Toutes ces données sont stockées uniquement sur l'appareil (IndexedDB du navigateur ou du WebView). Elles ne sont jamais transmises à un serveur ni à des tiers.

3. Collecte, utilisation et partage

Chrono EPS ne collecte, n'utilise ni ne partage aucune donnée avec des serveurs distants ou des tiers. Conformément à cette politique :

  • aucune collecte de données personnelles ou sensibles vers nos serveurs ;
  • aucun outil d'analyse (analytics) ou de suivi (tracking) ;
  • aucune transmission à des tiers ;
  • aucun partenaire n'a accès aux données.

4. Sécurité des données

Les données ne quittent jamais l'appareil. Elles sont stockées localement dans le stockage du navigateur (IndexedDB), géré par le système d'exploitation de l'appareil. Il n'existe pas de serveur distant sur lequel des données pourraient être stockées ou exposées. La suppression des données de l'application efface définitivement toutes les informations stockées.

5. Politique de conservation et de suppression

Conservation : Les données restent sur votre appareil tant que vous les conservez. Aucune donnée n'est conservée sur nos serveurs.

Suppression : Vous pouvez supprimer vos données à tout moment :

  • supprimer des courses ou des données depuis l'application ;
  • effacer toutes les données via les paramètres de votre appareil (données de l'app) ;
  • désinstaller l'application (supprime toutes les données locales).

Chrono EPS ne conserve aucune copie des données après leur suppression sur l'appareil.

6. Applications ciblant les enfants

L'application peut être utilisée par des enseignants dans un contexte scolaire impliquant des élèves de moins de 13 ans. Chrono EPS n'accède à aucune donnée utilisateur sensible transmise vers des serveurs. Les données saisies (noms d'élèves, performances) restent strictement sur l'appareil de l'enseignant, sans collecte ni transmission. Les applications qui n'accèdent à aucune donnée utilisateur sensible transmise à distance doivent néanmoins inclure des règles de confidentialité — la présente politique répond à cette exigence.

7. Contact

Pour toute question concernant cette politique de confidentialité ou l'exercice de vos droits : ouvrez une issue sur le dépôt GitHub du projet. Chrono EPS est un projet open source maintenu via le dépôt plamarque/chrono-eps.
